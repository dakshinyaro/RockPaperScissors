🎮 Rock Paper Scissors Game (Python GUI)

📌 Overview

This project is a **Rock Paper Scissors** game developed using **Python** and the **Tkinter** library. It provides an interactive graphical user interface (GUI) where users can play against the computer while tracking scores in real time.

✨ Features

* 🖥️ User-friendly GUI built with Tkinter
* 🎲 Random computer move generation using Python's `random` module
* 🪨 Rock, 📄 Paper, and ✂️ Scissors selection buttons
* 🏆 Automatic winner determination based on standard game rules
* 📊 Live score tracking for both the player and the computer
* 🔄 Reset button to restart the score at any time
* 🚫 Fixed-size application window for a consistent experience

🛠️ Technologies Used

* Python 3
* Tkinter (GUI Library)
* Random Module

▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/dakshinyaro/RockPaperScissors.git
```

2. Navigate to the project folder:

```bash
cd RockPaperScissors
```

3. Run the program:

```bash
python main.py
```

🎮 How to Play

1. Launch the application.
2. Click **Rock**, **Paper**, or **Scissors**.
3. The computer randomly selects its move.
4. The game compares both choices and determines the winner.
5. Scores are updated automatically after each round.
6. Use the **Reset Score** button to start over.
